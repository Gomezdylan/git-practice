## Description
RetroHockey is a fast-paced 2D arcade-style hockey game. Players control a skater on a 2D rink with shooting, passing, and defending to prevent goals against an AI or opposing player. With simple controls and smooth physics integrated by Unity, the game is designed for quick and fun matches.

---

## Controls

| **Action**             | **Keyboard Input** |
|------------------------|--------------------|
| Move Left              | ← Arrow            |
| Move Right             | → Arrow            |
| Move Up                | ↑ Arrow            |
| Move Down              | ↓ Arrow            |
| Wrist Shot / Slap Shot | **Spacebar**       |
| Pass / Release Puck    | **P**              |
| Slap Shot (Hold)       | **Hold Spacebar**  |
| Pause Game             | **Esc**            |
